# Knight Challenge

O desafio Knights visa a construção um sistema cadastro de heróis utilizando as
seguintes tecnologias:
• Vue.js
• NodeJS (Express.js | NestJS) | C# (.NET 7+)
• MongoDB | Redis
• Serviços emcontainers (script para build e geração)
